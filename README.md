# Portfolio Site

Static portfolio shell implementation for the portfolio direction in this workspace.

## Files

- `public/index.html`
  Single-page site entry.
- `public/styles.css`
  Visual system, layout, responsive behavior, and popup styling.
- `public/content.js`
  Structured placeholder content for sections and project popups.
- `public/app.js`
  Rendering, navigation state, popup behavior, and subtle reveal logic.

## Local Preview

From `portfolio/site`:

```bash
python3 -m http.server 4173 -d public
```

Then open `http://localhost:4173`.
